# NASA Explorer 🚀

An interactive space exploration app built with vanilla JavaScript and NASA's official APIs.

## Features
-  Astronomy Picture of the Day (APOD)
-  Earth imagery from space (EPIC)
-  Near Earth Objects tracker (NeoWs)
- Collapsible sidebar navigation

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- NASA Open APIs

## Run Locally
1. Clone the repo
2. Get a free API key at [api.nasa.gov](https://api.nasa.gov)
3. Add your key in `js/config.js`
4. Open `index.html` with Live Server
